## Occult Creations

Occult Creations bridges the gap between the wonderful worlds of Create and Occultism! This mod adds a touch of mystical automation to your Minecraft experience.

### Features

* **Unleash the Spiritwind!**  Infuse your Create fans with the power of Spiritfire from Occultism for a boost of otherworldly power.
* **Occultism Cranks for Create:**  Let the spirits do the work!  Use Occultism's Spirit Cranks to automate your Create contraptions through hand crank power.

### Dependencies

* Occultism ([https://www.curseforge.com/minecraft/mc-mods/occultism](https://www.curseforge.com/minecraft/mc-mods/occultism))
* Create ([https://www.curseforge.com/minecraft/mc-mods/create](https://www.curseforge.com/minecraft/mc-mods/create))

### Spirit Cranker Model & Texture

The model and texture for the Occultism Spirit Cranker used in this mod is a modified version of the original Occultism asset.

### Installation

1. Download the Occult Creations mod (place the .jar file in your mods folder).
2. Install the required dependencies (Occultism and Create).
3. Enjoy the mystical automation!
